# Campanha de Conscientização – UBS

Após entrevista com funcionários da UBS x x de suas necessidades, decidiu-se pela criação de uma campanha de conscientização, contando com um site informativo que contivesse o conteúdo que desejavam que fosse transmitido.

---

## Site informativo

O site tem compatibilidade tanto mobile e desktop, com funcionalidade de cache, para evitar uso excessivo de tráfego ao baixar o conteúdo, e ajustando para resolução de leitura perfeita, independente de qual dispositivo fosse usado para acessar.

## Página de administrador

Para que todos os integrantes do grupo, que haviam idades variadas, pudessem preencher e alterar o conteúdo do site conforme quisessem, foi escolhido o formato PDF para o conteúdo do site, e foi feita uma página de administrador. O projeto não havia back-end real, porém, com acesso a um token de api do repositório que havia sido disponibilizado ao grupo, podiam alterar a coleção de PDFs.
